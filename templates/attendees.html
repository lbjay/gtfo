This page is meant to demonstrate the potential of linked data for building distributed, collaborative databases on the web. The attendees table below is generated by a linked data web crawler that looks for FOAF profiles indicating that the subject is an attendee of $settings.EVENT_NAME.

If you want to play around with the data you can download a manageable snapshot of all the collected FOAF data. You might also want to look at the (dynamic) GraphViz views (image/png, application/postscript, image/svg+xml)

Want to see yourself here?

If you'd like to see yourself added to the list above:

  1. publish a FOAF file. Don't want to hand code? Try out foaf-a-matic and run your results through the validator; also you might want to try using RDFa to embed FAF assertions into an HTML page [example]
  2. assert in your FOAF file that you have attended $settings.EVENT_NAME
  3. have someone in the list above assert in their FOAF file that they know you
  4. wait for the next crawl to run
  5. profit! (not really)
